# 🧮  Calculator

A modern, responsive, and professional Calculator Web Application built using **HTML5, CSS3, and Vanilla JavaScript**.

It features a premium Glassmorphism UI, keyboard support, smooth animations, and a fully responsive layout that works across Desktop, Tablet, and Mobile devices.

## ✨ Features

- Modern Premium Glassmorphism UI
- Dark Theme
- Responsive Design
- Mobile Friendly
- Addition
- Subtraction
- Multiplication
- Division
- Percentage
- Decimal Support
- AC (All Clear)
- DEL (Delete)
- Error Handling
- Prevents Invalid Expressions
- Keyboard Support
- Enter Key = Calculate
- Backspace = Delete
- Escape = Clear
- Smooth Hover Animations
- Button Click Animation
- Ripple Effect
- Fade-in Animation
- Clean and Beginner-Friendly Code

## 💻 How to Run

1. Download or Clone this repository.

```bash

```

2. Open the project folder.

3. Open **index.html** in your browser.

No installation required.
